# 🏠 МійДім ОСББ — Веб-додаток

Простий, сучасний веб-додаток для управління ОСББ (Об'єднанням Співвласників Багатоквартирного Будинку).

## ✨ Можливості

- **Реєстрація та авторизація** мешканців і управителів
- **Особистий кабінет** з оглядом боргів та заявок
- **Новини та оголошення** від адміністратора
- **Заявки на ремонт** з фільтрацією та статусами
- **База мешканців** з пошуком
- **Деталізація платежів** та їх підтвердження
- **Адаптивний дизайн** для всіх пристроїв

## 🗂️ Структура

```
osbb-app/
├── index.html        # Головна/лендінг
├── register.html     # Реєстрація
├── login.html        # Вхід
├── dashboard.html    # Особистий кабінет
├── news.html         # Новини та оголошення
├── residents.html    # База мешканців
├── payments.html     # Оплата та нарахування
├── requests.html     # Заявки на ремонт
├── style.css         # Весь CSS (з градієнтами)
└── app.js            # Логіка + localStorage DB
```

## 🚀 Запуск

Просто відкрийте `index.html` у браузері. Або використайте Live Server у VS Code.

### GitHub Pages
1. Завантажте файли у репозиторій
2. `Settings → Pages → Source: main / root`
3. Готово! Сайт доступний за посиланням GitHub Pages

## 💾 База даних

Всі дані зберігаються у `localStorage` браузера. Для production-версії необхідно підключити бекенд (Firebase, Supabase або власний сервер).

## 🎨 Технології

- **HTML5, CSS3, Vanilla JS** — без фреймворків
- **Google Fonts** (Geologica + Inter)
- **CSS Grid + Flexbox** для адаптивності
- **localStorage** для зберігання даних

## 📱 Адаптивність

- Desktop: повний sidebar + 2-колонковий грід
- Tablet: collapsed sidebar
- Mobile: burger menu + single column
